Passo-a-passo da Fase 2:

- Adicionar repositório remoto
$ git remote add repo https://github.com/[URL]

ou

$ git clone https://github.com/[URL]

- Modificar arquivo existente
$ notepad README.md

- Verificar status
$ git status

- Adicionar ao repositório
$ git add README.md

- Verificar status
$ git status

- Commitar mudanças
$ git commit -m 'adiciona informações sobre autor'

- Verificar status
$ git status

- Enviar para repositório remoto
$ git push origin master
