Passo-a-passo da Fase 3:

- Criar um ramo a partir do ramo principal
$ git checkout -b fase3

- Criar novo arquivo
$ notepad monitoria.txt

- Verificar status
$ git status

- Adicionar ao repositório
$ git add monitoria.txt

- Verificar status
$ git status

- Commitar mudanças
$ git commit -m 'adiciona arquivo sobre a monitoria'

- Verificar status
$ git status

- Enviar para repositório remoto
$ git push origin fase03

- Modificar arquivo existente
$ notepad README.md

- Verificar status
$ git status

- Adicionar mudanças
$ git add README.md

- Commitar mudanças
$ git commit -m 'adiciona texto sobre contato'

- Juntar com ramo principal
$ git merge master
